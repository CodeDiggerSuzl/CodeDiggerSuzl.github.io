Get to Learn Vim

in order to be a an awesome ‘geek’, i’d like to learn the ancient editor. :/ this is gonna be cool :D

edit

  1. i insert
  2. a append
  3. o open a line below
  4. I INSERT before line
  5. O append before line

  6. A APPEND AFTER LINE

  7. shift + zz

  8. :wq write and quit

modes

  • normal mode
  • insert mode
  • command mode
  • visual mode

in insert mode

  • ctrl + h to delete a char
  • ctrl + w to delete a word
  • crtl + u to delete a line

These three shotcuts also works in shell or a terminal! crtl + a to head crtl + e to end crtl + b to before crtl + f to forward

quick to switch insert and normal mode

  • ctrl + c or ctrl + [ to quit insert and enter normal mode
  • gi to get into the last time you edit and enter into insert mode

quick move

  1. hjkl to move
  2. w/W to next word/WORD’s head
  3. e/E next word/WORD’s end
  4. b/B to next word/WORD’s head ; backword

serch in line

  • f{char} to move to char, t to the before char
  • use ; or , to get to next / previous
  • F to search front

move in line

  • use 0 to the first char of a line and ^ to the first char which is not a blank
  • user $ to the end of a line
  • 0 and $

move in page

  • gg/G to the begin/end of a file
  • use ctrl + o return back
  • H/M/L to head middle or lower of a page
  • ctrl+u/ ctrl+f upword/ forward
  • zz to put the screen to the middle

CRUD

INSERT

DELETE (d x)

in normal mode - x to delete a char - d(delete) to delete a word daw(d around a word) diw - d and x can combine with number to excute many times - u undo - dd delete a line - dt+{char} delete to char - d+$ - 2dd means delete 2 lines

fast update

r (replace)

in normal mode use r to replace a char and use s to delete word and enter into insert mode

c (change)

c delete and into insert

s (substitute)

query

  • use / or ? to search backward and forward
  • use n/N to jump to the next match
  • use * or # to match previous or next
