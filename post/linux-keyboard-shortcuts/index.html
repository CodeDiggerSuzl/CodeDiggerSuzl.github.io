Keyboards shortcuts for bash

线程相关

下面的快捷键能是你操作线程：

  1. Ctrl + C: 杀死终端中前台进行的线程，发送 SIGINT 信号到进程中, 从技术上讲，这只是一个请求 - 大多数进程都会接受它，但是有些可能会忽略它。
  2. Ctrl + Z: 挂起以 bash 运行的当前前台进程。 这会将 SIGTSTP 信号发送到该进程。 要在以后将进程返回到前台，请使用 fg process_name 命令。
  3. Ctrl + D: 关闭终端，发送 EOF 到 bash，bash 接受后就会退出，相当于 exit 命令。

控制屏幕相关的命令

  1. Ctrl + L: 相当于 clear 命令。
  2. Ctrl + S: 停止所有输出到屏幕。 这在运行带有大量冗长且冗长输出的命令时特别有用，但您不想使用 Ctrl + C 停止命令本身。
  3. Ctrl + Q: 继续 Ctrl+S 后的输入。

光标移动

  1. Ctrl + A 或 Home：转到该行的开头。
  2. Ctrl + E 或 End：转到行尾。
  3. Alt + B：向左（向后）一个字。
  4. Ctrl + B：向左（返回）一个字符。
  5. Alt + F：向右（前进）一个字。
  6. Ctrl + F：向右（前进）一个字符。
  7. Ctrl + XX：在行的开头和光标的当前位置之间移动。 这使您可以按 Ctrl + XX 返回到行的开头，更改某些内容，然后按 Ctrl + XX 返回到原始光标位置。 要使用此快捷方式，请按住 Ctrl 键并点击 X 键两次。

删除文本

  1. Ctrl + D: 或 Delete：删除光标下方的字符。
  2. Alt + D：删除当前行上光标之后的所有字符。
  3. Ctrl + H 或 Backspace：删除光标之前的字符。

修复错字

  1. Alt + T：将当前单词与上一个单词交换。
  2. Ctrl + T：将光标前的最后两个字符彼此交换。 当您以错误的顺序键入两个字符时，可以使用它来快速纠正错字。
  3. Ctrl + _：撤消上一次按键操作。 您可以重复此操作以撤消多次。

剪切和粘贴

  1. Ctrl + W：在光标之前剪切一个单词，然后将其添加到剪贴板。
  2. Ctrl + K：剪切光标后的部分行，将其添加到剪贴板。
  3. Ctrl + U：剪切光标之前的部分行，将其添加到剪贴板。
  4. Ctrl + Y：粘贴从剪贴板剪切的最后一件事。 此处的 y 代表 “yank”。

大写字符

bash shell 可以快速将字符转换为大写或小写：

  1. Alt + U：大写从光标到当前单词结尾的每个字符，将字符转换为大写。
  2. Alt + L：对从光标到当前单词结尾的每个字符取消大写，将字符转换为小写。
  3. Alt + C：大写光标下的字符。 您的光标将移动到当前单词的末尾。

Tab 补全

Tab：自动完成您输入的文件，目录或命令。

历史相关

  1. Ctrl + P 或向上箭头：转到命令历史记录中的上一个命令。 多次按快捷方式可追溯历史记录。
  2. Ctrl + N 或向下箭头：转到命令历史记录中的下一个命令。 多次按快捷方式可向前浏览历史记录。
  3. Alt + R：如果您已对其进行了编辑，则将所有更改恢复到您从历史记录中拉出的命令。

Bash 还具有一种特殊的 “调用” 模式，可用于搜索以前运行的命令：

  1. Ctrl + R –在 bash 历史记录中开始反向搜索，只需键入要在历史记录中查找的命令应唯一的字符。
  2. Ctrl + S –通过 bash 历史记录启动向前搜索。
  3. Ctrl + G –在 bash 历史记录中退出反向或向前搜索。
  4. Ctrl + O：运行使用 Ctrl + R 找到的命令。

Bash Bang(!) 命令

  1. !!: 执行最后一条命令。
  2. !top: 执行以'top'开头的最新命令（例如！）。
  3. !top:p: 显示 !top 将运行的命令（还将其添加为命令历史记录中的最新命令）。
  4. !$: 执行上一个命令的最后一个单词（与 Alt + . 相同，例如，如果最后一个命令是’cat tecmint.txt’，那么！$ 将尝试运行’tecmint.txt’）。
  5. !$:p–显示 !$ 将执行的单词。
  6. !* –显示上一个命令的最后一个单词。
  7. !*:p–显示 !* 替代的最后一个单词。

emacs vs. vi 键盘快捷键

以上说明假定您使用的是 bash 中的默认键盘快捷键配置。 默认情况下，bash 使用 emacs 样式的键。 如果您更习惯于 vi 文本编辑器，则可以切换到 vi 样式的键盘快捷键。

以下命令将使 bash 进入 vi 模式：

1

set -o vi

以下命令会将 bash 放回默认的 emacs 模式：

1

set -o emacs
