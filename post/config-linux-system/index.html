Config Manjaro(Brand New Linux System)

For future config back and other intentions.

Not hand by hand, just url Config in Manjaro

First of all ：git config

  1. git config –gloable user.name “username”
  2. git config –gloable user.email “email”
  3. using ssh-keygen -t ras to generate ssh pub-keys, paste in to github settings in order to pull&&push code.

Shell Config

  1. zsh && oh-my-zsh

    1. using curl

      sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

    2. using wget

      sh -c "$(wget https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh -O -)"

  2. autojump

    in manjaro linux:

    1. sudo pacman -S autojump

  3. fzf

    sudo pacman -S fzf

  4. tmux

    sudo pacman -S tmux

  5. vim&&nvim

    sync your .vimrc

  6. gitmoji

    1. pacman -S npm
    2. npm i -g gitmoji-cli`

  7. tldr(too long don’t read)

    1. pacman -S tldr
