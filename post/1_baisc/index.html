类型转换

Go语言中不允许隐式转换，所有类型转换必须显式声明（强制转换），而且转换只能发生在两种相互兼容的类型之间

fmt

格式 含义
%c 字符型。可以把输入的数字按照 ASCII 码相应转换为对应的字符
%d 一个十进制数值 (基数为 10)
%f 以标准记数法表示的浮点数或者复数值
%s 字符串。输出字符串中的字符直至字符串中的空字符（字符串以’\0‘结尾，这个’\0'即空字符）
%b 一个二进制整数值 (基数为 2)，或者是一个(高级的) 用科学计数法表示的指数为 2 的浮点数
%t 以 true 或者 false 输出的布尔值
%T 使用 Go 语法输出的值的类型
%% 一个%字面量
%o 一个以八进制表示的数字(基数为8)
%p 以十六进制(基数为16)表示的一个值的地址，前缀为0x,字母使用小写的a-f表示
%x(X) 以十六进制表示的整型值(基数为十六)，数字a-f使用小(大)写表示

iota 枚举

常量声明可以使用 iota 常量生成器初始化，它用于生成一组以相似规则初始化的常量，但是不用每行都写一遍初始化表达式。 注意：在一个 const 声明语句中，在第一个声明的常量所在的行，iota 将会被置为 0，然后在每一个有常量声明的行加 1

切片

定义:

  1. 空切片:s:=[]int{}
  2. var s1 []int没有长度
  3. 借助make函数, 格式 make(切片类型, 长度, 容量) s := make([]int ,5,10)
    1. 切片的长度要小于容量
    2. len(s) return len
    3. cap(s) return cap
  4. 循环赋值的结束条件是小于切片的长度，而不是容量
  5. s[low:high:max]
    1. low startIndex
    2. high endIndex
    3. max 所谓容量:是指切片目前可容纳的最多元素个数。通过公式(cap=max-low)计算
  6. 使用 append (s,element)进行添加
  7. copy(slice1,slice2) 将第二个切片里面的元素，拷贝到第一个切片中。

字符串 使用strings 包中的值

  1. funcContains(s,substrstring)bool

    功能：字符串s中是否包含substr，返回bool值

  2. funcJoin(a[]string,sepstring)string

    功能：字符串链接，把slicea通过sep链接起来

  3. funcIndex(s,sepstring)int

    功能：在字符串s中查找sep所在的位置，返回位置值，找不到返回-1

  4. funcReplace(s,old,newstring,nint)string

    功能：在s字符串中，把old字符串替换为new字符串，n表示替换的次数，小于0表示全部替换

  5. funcSplit(s,sepstring)[]string

    功能：把s字符串按照sep分割，返回slice

  6. funcTrim(sstring,cutsetstring)string

    功能：在s字符串的头部和尾部去除cutset指定的字符串

  7. funcRepeat(sstring,countint)string

    功能：重复s字符串count次，最后返回重复的字符串

  8. funcFields(sstring)[]string

    功能：去除s字符串的空格符，并且按照空格分割返回slice
