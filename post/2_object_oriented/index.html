函数和方法的区别

  1. 在关键字后面加上( a Integer), 这个在方法中称之为接收者，所谓的接受者就是接收传递过来的第一个参数，然后复制a， a的类型是Integer ,由于Integer是int的别名，所以a的类型为int

  2. 在表示参数的类型时，都使用了对应的别名。

    通过方法的定义，可以看出方法其实就是给某个类型绑定的函数。在这里，是为整型绑定的函数，只不过在给整型绑定函数(方法)时，一定要通过type来指定一个别名，因为int类型是系统已经规定好了，无法直接绑定函数，所以只能通过别名的方式。

  3. 调用方式不同

    var result Interger=3 表示定义一个整型变量 result，并赋值为 3.

    result.Sum(3)

    通过result变量，完成方法的调用。因为，Sum( )方法，是为int类型绑定的函数，而result变量为int类型。所以可以调用 Sum( )方法。result变量的值会传递给 Sum( )方法的接受者，也就是参数a,而实参Sum(3),会传递形参b. 当然，我们也可以将Sum( )方法，理解成是为int类型扩展了，追加了的方法。因为系统在int类型时，是没有改方法的。

如果要修改对应的值,将方法的接收者，修改成对应的指针类型

注意事项

  1. 只要接收者类型不一样，这个方法就算同名，也是不同方法，不会出现重复定义函数的错误,如果接收者类型一样，但是方法的参数不一样，是会出现错误的。

Empty Interface

空接口(interface{})不包含任何的方法，正因为如此，所有的类型都实现了空接口，因此空接口可以存储任意类型的数值。

Type Assert

UAAGE:

value,ok = element.(T)

  1. value: the value of element
  2. ok: a bool type value
  3. element: an interface variable
  4. T: the type of assert
